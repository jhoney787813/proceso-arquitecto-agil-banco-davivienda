
# Proceso Selección Arquitecto Ágil - Banco Davivienda

El propósito del repositorio es evidencia diseñado para evidenciar los conocimientos y capacidades de un candidato para el puesto de **Arquitecto de Software Ágil**. En él se abordan tanto las decisiones técnicas clave como las buenas prácticas en el desarrollo de sistemas de software dentro de un contexto ágil.

El objetivo es plasmar las mejores prácticas, el entendimiento profundo de la arquitectura de software y los acuerdos de integración que deben ser aplicados para garantizar el éxito de la plataforma.
