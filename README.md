
# Proceso Selección Arquitecto Ágil - Banco Davivienda

El propósito del repositorio es evidenciar los conocimientos como **Arquitecto Ágil** en la construcción de proyectos para la compañía **Banco Davivienda**.

Este documento refleja las prácticas y definiciones de alto nivel, así como el enfoque integral no solo desde un punto de vista abstracto, sino también con énfasis en los **acuerdos de integración para el desarrollo** dentro de un entorno ágil y escalable.

---

## Test para el Puesto de Arquitecto de Software Ágil

Este es un test diseñado para evaluar los conocimientos y capacidades de un candidato para el puesto de **Arquitecto de Software Ágil**. En él se abordan tanto las decisiones técnicas clave como las buenas prácticas en el desarrollo de sistemas de software dentro de un contexto ágil.

El objetivo es plasmar las mejores prácticas, el entendimiento profundo de la arquitectura de software y los acuerdos de integración que deben ser aplicados para garantizar el éxito de la plataforma.
